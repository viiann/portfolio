Développeur·se web - Formation chez Simplon.Co Calais
Certification : Passage du titre professionnel RNCP de niveau III, équivalent Bac+2, “Développeur web et web mobile”
Les missions du Métier :
De l’analyse du besoin à la mise en ligne, en passant par l’interface et la base de données, le·la développeur·se web conçoit et programme des applications web.
analyse les besoins des clients consignés au préalable dans un cahier des charges par le chef de projet. préconise et met en œuvre une solution technique pour concevoir des sites sur mesure ou adapter des solutions techniques existantes.
Le métier de développeur·se Web s’articule alors autour de 2 activités principales :
Développer la partie front-end d’une application web ou web mobile en intégrant les recommandations de sécurité
Développer la partie back-end d’une application web ou web mobile en intégrant les recommandations de sécurité
